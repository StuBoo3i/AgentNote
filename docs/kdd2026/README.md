## 来源标记说明

为了让阅读者区分不同笔记来源，本栏目会在每篇笔记入口和正文顶部标注来源：

- `AI 生成`：由 AI 根据项目运行结果、代码结构或分析目标生成，后续可能需要人工复核。
- `人工整理`：由人手动总结、修改或长期维护。
- `混合整理`：人工初稿加AI润色，人工校对、补充和修订。
