# 大模型基础

## 学习目标

- 理解大语言模型的基本能力边界
- 理解 Token、上下文窗口、Embedding、Prompt 的作用
- 建立后续学习 RAG、Agent、微调和部署的基础

## 核心概念

### Token

Token 是模型处理文本的基本单位，可以近似理解为字、词或词片段。

### 上下文窗口

上下文窗口决定模型一次能读取和生成的文本总长度。上下文越长，能放入的历史信息越多，但成本和延迟通常也更高。

### Prompt

Prompt 是给模型的任务说明。好的 Prompt 应该明确目标、输入、输出格式和约束。
