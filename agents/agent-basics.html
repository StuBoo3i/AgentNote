# Agent基础

# Agent 是什么

Agent 可以理解为能够围绕目标进行观察、思考、调用工具并执行任务的智能系统。

# 常见组成

  • 模型：负责理解、推理和生成
  • 工具：负责访问外部能力，例如搜索、代码执行、数据库、文件系统
  • 记忆：负责保存历史信息、用户偏好和任务状态
  • 规划：负责拆解目标、选择步骤并根据结果调整行动

# 学习路线

  1. 学会 Prompt 和结构化输出
  2. 学会函数调用和工具调用
  3. 学会 RAG 和外部知识检索
  4. 学会多步骤任务规划
  5. 学会评测 Agent 是否可靠
